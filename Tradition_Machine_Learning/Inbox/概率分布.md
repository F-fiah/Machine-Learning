# 指数家族

$$ p(y\mid \eta)=b(y)\exp\big(\eta^\mathrm{T}T(y)-A(\eta)\big) $$

- $y$ : data
- $\eta$ : natural parameter 常见的为 $\eta = \theta^T x$
- $b(y)$ : base measure 固定基准项 不含有 $\eta$
- $A(\eta)$ : 对数配分函数，只含有 $\eta$
- $T(y)$ : 充分统计量，从样本y里提取出能代表全部信息的统计值  通常 $T(y) = y$

**指数家族的advantages：**
1. 对数似然函数是凹函数
2. 配分函数一阶导 = y 的期望 即 $\mathbb{E}[y\mid \eta]=\frac{\partial A(\eta)}{\partial \eta}$
3. 配分函数二阶导 = y 的方差 即 $\mathrm{Var}[y\mid \eta]=\frac{\partial^2 A(\eta)}{\partial \eta^2}$
## 伯努利分布（二元数据）
$$ P(y;\phi)=\phi^y(1-\phi)^{1-y},\quad y\in\{0,1\} $$
$\phi$：$y=1$ 的概率
