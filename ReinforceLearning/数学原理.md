# Markov Process

## Probability
**The future is independent of the past given the present**

$$\mathcal{P}_{ss'} = \mathbb{P}\left[S_{t+1} = s' \mid S_t = s\right]$$
## Return
$$G_t = R_{t+1} + \gamma R_{t+2} + \dots = \sum_{k=0}^{\infty} \gamma^k R_{t+k+1}$$
$\gamma$ :  
- Mathematically convenient to discount rewards
- Avoids infinite returns
- preference for immediate reward
## Value
$$ \begin{aligned} v(s) &= \mathbb{E}\left[G_t \mid S_t = s\right] \\ &= \mathbb{E}\left[R_{t+1} + \gamma R_{t+2} + \gamma^2 R_{t+3} + \dots \mid S_t = s\right] \\ &= \mathbb{E}\left[R_{t+1} + \gamma\left(R_{t+2} + \gamma R_{t+3} + \dots\right) \mid S_t = s\right] \\ &= \mathbb{E}\left[R_{t+1} + \gamma G_{t+1} \mid S_t = s\right] \\ &= \mathbb{E}\left[R_{t+1} + \gamma v(S_{t+1}) \mid S_t = s\right] \end{aligned} $$
$R_{t+1}$ ：单步收益（即时奖励），$\gamma v(S_{t+1})$ ：未来收益折算成当前价值
$$v(s) = \mathcal{R}_s + \gamma \sum_{s' \in S} \mathcal{P}_{ss'} v(s')$$
## Action

在不同状态 s 下的行为

## Policy

A policy $\pi$ is a distribution over actions given states
- Policies depend on the current state (not the history)
- Policies are stationary

## Q

$$ Q^\pi(s,a) = \mathbb{E}_\pi \left[ G_t \mid S_t = s, A_t = a \right] $$
