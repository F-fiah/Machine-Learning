# 线性神经网络

线性神经网络的缺陷
- Linear classifiers learn one template per class
  线性分类器只能记住一类物体 “标准长相”，无法兼容同类别的不同形态（因为只有一层）
- Linear classifiers can only draw linear decision boundaries
  只能生成线性分割面，无法处理非线性交错的数据分布

# 卷积神经网络（convolutional neural network）

如果直接将整张图片的所有像素转换成一个一维的向量，会破坏掉图像的空间结构，从而损失有效信息
虽然理论上全连接网络确实有能力学习空间关系，但成本很大

![[CNN.png]]
## Build CNN
### Fully-Connected Layer 全连接层

Refer to [[Image Classification#Linear Classifier]]
### Activation Function 激活函数

- Sigmiod：large positive or negative values will kill the gradients
- ReLU：
	1. computationally efficient(converge fast)
	2. die when x<0
- GELU：
	1. nice behavior around 0 (smooth)
	2. higher computational cost
	3. large negative values will kill the gradients

Activations generally placed after linear operators 
### Convolution Layer 卷积层

![[filter.png]]
每一个 filter 对应一组 $(W,b)$，对于每一个截取到的卷积快，将这个像素块拉平成一维向量，线性转换成输出标量（卷积 = 滑动的点积）

若过滤器的维度为 k，则经过卷积后输出维度变为 k

为了往卷积层中加入非线性性，使用激活函数

卷积核 = 局部图像模板，和图像局部窗口做**点积相似度匹配**
当窗口内出现明暗分界（边缘）时，明暗、色彩突变区域会产生更大数值差异，能更快降低分类损失
So the first-layer conv filter often learns oriented edged and opposing colors（because it can only see pixels in its local region）
Instead, deeper conv layers tend to learn lager structures（the receptive field grows linearly with the number of convolution layers）

如何化初始化卷积核：随机初始化

经过卷积后，feature maps 的大小会缩小，因此可以在原 feature maps 的边缘扩充，使得卷积前后大小不变
扩充的像素值：
- 全部置零
- 将原图像的边缘进行复制

![[cnn-summary.png]]

**A larger set of smaller filters is more expressive than a smaller set of larger filters**
### Pooling Layer 池化层

池化层的处理本质上是做下采样（缩小特征图 / 图片的空间分辨率），从而加快运算
池化层通常和卷积层交替使用

- Max-Pooling：将图划分成多个小区域，每个区域用最大像素值代替
- Average-Pooling
- ……
池化操作可能会引入非线性性，这取决于操作的函数
## Train CNN
### Layer-Norm

1. 标准化：把输入数据规整到均值 0、方差 1，消除数值分布偏移
2. 缩放 + 平移：还原模型需要的数据分布
$$ y_{i,d} = \gamma_d \cdot \frac{x_{i,d} - \mu_i}{\sigma_i} + \beta_d $$
- 在卷积 / 全连接线性变换之后、激活函数之前进行 normalization
- 每一层都要进行，测试时直接使用训练时该层的 $\mu,\sigma$
- $\gamma,\beta$ 可以通过梯度下降进行学习
### Dropout（暂退法）

训练时随机删去一些神经元，测试时所有神经元均保留
### ResNet

**层数越大并不一定意味着模型越精准**
Deeper model has more representation power than shallower models, but it's harder to optimize(gradient vanish/boom)

**How to build a deeper model st least as good as the shallower model：ResNet**

假设一个 $N_1$ 层的网络效果很好，现在我们想做 $N_1 + N_2$ 层
将原有卷积层的输出 $x$ 分别传入两部分：
- 短路：原样传递，输出 $x$
- 新增卷积层 $N_2$，输出 $F(x)$
如果新增卷积层学习到有用信息，则最终融合 $H(x) = x + F(x)$，否则 $F(x) = 0, \quad H(x) = x$
### Weight Initialization

激活值的方差 $\mathrm{Var}(xW) = D_{\mathrm{in}} \cdot \mathrm{Var}(x) \cdot \mathrm{Var}(W)$

若给权重初始化乘极小常数，则层数越深激活的均值、标准差越趋近 0（神经元死亡 + 梯度消失）
若初始化乘大常数，则激活的均值、标准差数值爆炸（梯度爆炸）

**Kaiming Initialization：**
标准正态随机权重，每层单独独立初始化
$W = std \cdot \mathcal{N}(0, 1),\quad std = \sqrt{\frac{2}{D_{in}}}$，其中 $D_{in}$ 为当前层输入神经元数量
### Data Process/Augmentation

1. 对图像做水平/垂直翻转
2. 对图像进行随机裁剪/缩放
3. 颜色/亮度抖动
4. 随机掩盖图片的一部分

- 在神经网络训练时，每次迭代随机生成 1 张裁剪 / 翻转图，并只输入一张图进入网络，这样可以增加数据多样性，防止过拟合
- 在测试阶段，则一次性生成几十张衍生图，全部预测再平均，从而消除随机带来的预测波动，提升推理精度
### Transfer Learning

|                     | very similar dataset | very different dataset |
| ------------------- | -------------------- | ---------------------- |
| very little data    | 冻结全部卷积权重，仅训练末尾线性分类头  | 扩充数据集 / 更换适配领域预训练模型    |
| quite a lot of data | 全部网络层放开，整体微调         | 全层微调 / 从零开始完整训练        |
### Choose Hyperparameters

| 步骤            | 操作内容                          | 核心目的               | 判断标准                  |
| ------------- | ----------------------------- | ------------------ | --------------------- |
| Step1 检查初始损失  | 不训练，单次前向计算loss                | 校验损失函数、标签、输出层代码    | 初始loss接近随机猜测理论值       |
| Step2 小样本过拟合  | 少量样本，关闭全部正则，充分训练              | 验证网络结构、前向/反向传播无bug | 训练loss趋近0，训练准确率接近100% |
| Step3 寻找合适学习率 | 全量训练集，开启轻度weight decay，遍历候选LR | 找到收敛速度最优的基础学习率     | 约100迭代内loss显著下降       |
| Step4 粗网格超参搜索 | 大范围超参组合批量测试 | 快速淘汰劣质超参，缩小区间 | 1~5 epoch |
| Step5 精细网格搜索 | 在优质区间缩小范围完整训练 | 锁定全局最优超参组合 | 完整多epoch |
| Step6 分析损失&精度曲线 | 绘制训练/验证loss、acc曲线 | 诊断过拟合、欠拟合、收敛异常 | 完整训练完成后 |
欠拟合：
1. 延长训练时间，让损失充分下降
2. 增大模型容量：加深网络、增加通道数、加宽全连接层，提升拟合能力
3. 减弱 / 移除正则化约束：减小 weight decay/ dropout 概率
4. 调整优化器，加速收敛

过拟合：
1. 数据增强，增加数据多样性
2. 正则化
3. 降低模型容量
4. 扩充数据集

- 网络搜索超参：人为给两个超参分别设置固定离散取值，两两全部配对，逐个测试
- 随机搜索：在超参取值范围内均匀随机采样 N 组独立组合
# 循环神经网络（Reccurrent Neural Networks）

用于有效处理数据的序列信息
RNN 的输出不仅核当前的数据输入有关，海域过去的信息相关
$$\begin{align} h_t &= f_W(h_{t-1},x_t) \quad f_W与t无关 \\ y_t &= f_y(W_{hy}h_t) \end{align}$$
其中 $h_t$ 为输入与输出之间的隐状态，常用的函数为：
$\begin{align} h_t &= \tanh\left(W_{hh}h_{t-1} + W_{xh}x_t + b_h\right) \\ &= \tanh\left( W \begin{pmatrix} h_{t-1} \\ x_t \end{pmatrix} \right) \end{align}$

![[rnn-cell.png]]

总损失 $\mathcal{L} = \sum_{t=1}^{T} \mathcal{L}_t$，即每个时刻的损失的总和

![[RNN.png]]
Embedding 嵌入层：
  把离散字符映射到低维稠密向量，让语义相近字符向量距离更近，防止维度爆炸
$$ \begin{cases} x_{\mathrm{emb},t} = W_{\mathrm{emb}} \cdot x_{\mathrm{onehot},t} \\ h_t = \tanh\left(W_{hh}h_{t-1} + W_{xh}x_{\mathrm{emb},t}\right) \end{cases} $$
RNN 可以有多层隐状态，每一层使用单独的 W

Advantages
1. **支持任意长度序列输入**
2. **理论可复用久远历史信息**
    隐状态 $h_t$ 循环传递，每一步融合全部前文上下文
3. **参数量固定，不随序列变长膨胀** 
   所有时间步共享一套权重 $W_{xh},W_{hh},W_{hy}$，无维度爆炸
4. **时序平移对称性** 
   全局共享权重，相同时序规律无论出现在序列哪个位置都能同等识别

Disadvantages
1. **循环串行计算，速度很慢**
    计算 $h_t$ 必须等待 $h_{t-1}$ 完成，无法GPU并行加速，长序列训练耗时极高
2. **现实无法捕捉远距离依赖**
    tanh导数连续相乘导致**梯度消失**，序列开头信息无法传递到末尾
