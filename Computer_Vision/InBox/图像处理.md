# Semantic Segmentation

Label each pixel in a image with a semantic category

思路1：用一小块方形窗口在整张图逐像素滑动，使用 CNN 来判断截取的 patch 的 label，从而得出中心像素的 label，遍历所有像素
But it's really time consuming!!!

思路2：把整张完整图片一次性送入 CNN 提取全局特征，直接在网络末端完成语义分割
但 CNN 会损失物体边缘、细小像素等信息（pooling layer+downsampling operator），行不通

思路3：设计一个 CNN，只有卷积层，不采用池化和下采样操作
But it's computatively expensive

思路4：downsample 后重新 uosample，先压缩提取全局特征，再放大还原到原图尺寸输出分割图
## Fully Convolutional Nerual Network
### Upsampling

为了实现逐像素预测，必须用 upsample 把特征图把特征图变回原图的尺寸 

1. 近邻插值
   新像素直接复制距离最近的原有特征像素的值
2. 双线性插值
   新像素由周边 4 个原始像素加权平均计算得出
3. 反卷积
   将原始像素作用于一个反卷积矩阵，成为新的多个像素点
# Object Detection

对于单物体检测，直接使用 CNN 训练即可
对于多物体检测，可与拆分成多个单物体检测，每一个都训练一个独立的 CNN，但极度耗费算力
## R-CNN
### Slow R-CNN

1. 根据原图生成候选框
2. 每一个候选框图进行 CNN
### Fast R-CNN

1. 整张原图进行一次 CNN 卷积
2. 将候选框映射到全局特征图上，在原特征图上抠出小块特征 Rol，并池化成统一尺寸
3. 将 Rol 特征送入全连接 FC 网络，进行 objet category+box offset
## Transformers

- 使用 Transformers 可以很好地处理上下文信息
- Transformer 底层只处理向量序列，不区分文本 / 图像，所以要先通过 CNN 将图像展平乘一个特征向量
- 图像没有天然时序顺序，需要通过位置编码补齐图像空间信息

1. 将原图送入 CNN，下采样得到图像特征图，同时叠加上位置编码
2. 将图像特征图传入 Transformer 编码器，提取全局图像上下文特征
3. 将 Encoder 输出的全局图像特征传入 Transformer 解码器，结合物体查询向量，输出结果向量
4. 将结果向量传入 MLP，得到 class + box 的输出
# Instance Segmentation
## Mask R-CNN

Fast R-CNN + Mask 预测

Faster R-CNN 只能输出形边界框和物体类别，无法得到物体精确像素轮廓
因此在 Faster R-CNN 的基础上，新开一条卷积网络，进行掩码

1. 将 Rol 特征图传入
2. 通过卷积把图的空间尺寸还原乘原框图的尺寸，每个通道里是一张黑白掩码
3. 训练时，只计算对应类别的损失
4. 将掩码图还原乘原图
